# PaintStore_backEnd

To configure add appsettings.json file with this text,

{
  "ConnectionStrings": {
    "PaintStoreDatabase": "<conection string>"
  },
  "Logging": {
    "IncludeScopes": false,
    "Debug": {
      "LogLevel": {
        "Default": "Warning"
      }
    },
    "Console": {
      "LogLevel": {
        "Default": "Warning"
      }
    }
  }
}
  
<connection string> there past your connection string
